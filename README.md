The goal of this project is to train an intelligent agent that learns how to play Blackjack by interacting with the environment
and maximizing its expected reward over time — without being explicitly programmed with Blackjack strategy rules.

The agent improves its decision-making through trial and error using reinforcement learning algorithms.


The project explores key RL concepts such as:
1.Markov Decision Process (MDP)
2.State–Action–Reward framework
3.Exploration vs Exploitation trade-off
4.Policy improvement over episodes

Depending on your implementation, you can mention:
1.Q-Learning
2.Monte Carlo Control
3.SARSA
4.Deep Q-Network (DQN)
